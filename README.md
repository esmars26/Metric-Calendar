# Metric-Calendar
Changes the current time (UTC) into metric time.\
Metric time is based on the standardized base-10 system, like the metric measurement system.
> [!NOTE]
> This converter uses an [epoch](https://en.wikipedia.org/wiki/Epoch) of 2000-01-01T00:00:00Z, or Sat, 01 Jan 2000 00:00:00 UTC/GMT
